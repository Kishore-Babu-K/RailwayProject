Project Overview

Railway Enquiry System is an intuitive and user-friendly web-based application designed to streamline the process of railway information retrieval. This system provides passengers with the ability to search for trains based on train numbers, train names, or stations. It displays detailed train schedules, including the train number, name, available routes, and running dates, helping users plan their journeys with ease. The system also supports user authentication for personalized access, allowing registered users to manage their information securely. With an interactive interface and real-time data, the Railway Enquiry System makes accessing crucial railway information fast and convenient, enhancing the overall travel experience.

Key Features

  • Train Search: Ability to search for trains by train number or name, as well as by source and destination stations.
  • User Authentication: Registered users can log in to manage personal data and receive tailored services.
  • Interactive Interface: Easy-to-navigate interface that simplifies the process of train information retrieval.

Technology Stack

  • Frontend: HTML, CSS, Bootstrap, and JavaScript for responsive and interactive design.
  • Backend: Flask web framework for handling routes, user requests, and server-side logic.
  • Database: MySql database for storing train data, schedules, and user details.
  • Additional Libraries: Jinja for rendering templates, ast.literal_eval for handling data parsing.
